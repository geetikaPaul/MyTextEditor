Basic text editor built for fun.

Language used: Java
Packages used:javax.swing, java.awt

Functionalities included:
New
Open - in new window or current window
Save
Wordwrap

Keyboard shortcuts supported for all the functionalities.

How to run in local:
install Java
Clone the project
Go to src folder
To build the project: run _javac *java -d ._
To run the project: _java Main_
